# BMI Calculator

A web app built with Flask that calculates your BMI based on your weight and height.

## Live Demo
[View Live App](https://your-render-url.onrender.com)

## Features
- Calculate BMI using weight (lbs) and height (feet/inches)
- Displays BMI result and weight category
- Input validation and error handling
- Clean responsive design

## Tech Stack
- Python
- Flask
- Jinja2 templates
- Deployed on Render

## Setup
1. Clone the repo
   git clone https://github.com/YOURUSERNAME/bmi-calculator.git

2. Install dependencies
   pip install -r requirements.txt

3. Set environment variable
   export SECRET_KEY=your-secret-key

4. Run the app
   python app.py

5. Open http://localhost:5000 in your browser
